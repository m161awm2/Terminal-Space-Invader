# 🚀 Terminal Space Invader (Python Curses)

리눅스 터미널에서 즐기는 정통 슈팅 게임입니다. `curses` 라이브러리를 사용하여 외부 그래픽 엔진 없이 텍스트만으로 박진감 넘치는 액션을 구현했습니다.

## 🕹️ 게임 특징
* **다양한 아이템**: Multishot(분산 사격), Railgun(관통 레이저) 시스템.
* **성장 시스템**: 라운드 클리어 시 발사 수량 및 재생 속도 업그레이드.
* **보스전**: 10라운드마다 강력한 패턴을 가진 보스 등장.

## 🛠️ 설치 및 실행 방법
리눅스 환경(또는 macOS)에서 별도의 라이브러리 설치 없이 실행 가능합니다.

```bash
# 저장소 클론
git clone [https://github.com/사용자아이디/저장소이름.git](https://github.com/사용자아이디/저장소이름.git)
cd 저장소이름

# 게임 실행
python3 shooting_game.py
